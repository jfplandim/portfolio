# 💻 Meu Portfólio Profissional

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido inicialmente como parte da disciplina de **Programação Web** do curso de Engenharia de Software no **iCEV**, com o objetivo de consolidar e expor minhas habilidades técnicas, formação acadêmica e principais projetos práticos.

O site é totalmente responsivo, possui uma identidade visual limpa e moderna focada na experiência do usuário, além de contar com navegação otimizada e dinâmica.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído puramente com tecnologias web nativas (*Vanilla Web Development*), demonstrando o domínio dos fundamentos de front-end:

* **HTML5:** Estruturação semântica de todo o conteúdo da página.
* **CSS3:** Estilização moderna utilizando **Flexbox** e **Grid Layout** para garantir total responsividade em dispositivos móveis, tablets e desktops. Conta com transições suaves e design simétrico.
* **JavaScript (ES6+):** Lógica para manipulação do DOM, incluindo scroll suave nativo e o efeito dinâmico de destaque na barra de navegação à medida que o usuário explora a página.

---

## 📂 Principais Projetos Destacados

No portfólio, apresento uma vitrine de projetos que abrangem desde o desenvolvimento de interfaces web até lógica complexa de sistemas e algoritmos:

1.  **Syntax — Curso de Programação Web:** Landing page responsiva com integração à API do GitHub, sistema de matrículas via Mercado Pago e design baseado na paleta *Catppuccin Mocha*.
2.  **Escalonador de Processos — iCEVOS:** Simulador de sistemas operacionais desenvolvido do zero em Java (sem bibliotecas externas) para gerenciar filas de prioridade e prevenção de inanição (*starvation*).
3.  **Sistema de Administração de Condomínios — Vista Alegre:** Sistema robusto em Java aplicando os pilares avançados de Programação Orientada a Objetos (POO) com tratamento de exceções customizadas e persistência de dados em arquivos TXT.
4.  **Sistema de Análise Forense Computacional:** Ferramenta desenvolvida em Java em formato `.jar` focada no processamento estruturado de dados, auditoria de integridade de arquivos e identificação automatizada de padrões forenses.

## 🛠️ Como Executar o Projeto Localmente

Para clonar e rodar este portfólio na sua máquina, você não precisa de nenhuma dependência ou servidor pesado instalado. Basta seguir os passos:

1.  **Clone este repositório:**
    ```bash
    git clone https://github.com/jfplandim/portfolio.git
    ```
2.  **Navegue até a pasta do projeto:**
    ```bash
    cd portfolio
    ```
3.  **Abra o arquivo principal:**
    Basta dar um duplo clique no arquivo `index.html` ou abri-lo diretamente através do seu navegador de preferência (Chrome, Edge, Firefox, Brave, etc.).

*Dica de Dev: Se você utiliza o VS Code, pode usar a extensão **Live Server** para rodar o projeto com recarregamento automático a cada alteração de código.*
